### 🧠 Engineering Philosophy
> *"Control framework > model size."*

- **Production First:** Jupyter notebooks are for exploration; modular `.py` files are for shipping.
- **Efficiency over Hype:** Focused on localized subgraph repair, narrow contexts, and avoiding token bloat.
- **Architectural Control:** Designing explicit triggers and evaluators instead of letting the LLM guess the next move.
